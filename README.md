# Gelson IA 2.0 - Browser AI Studio for Creators 🚀

**Edit video with AI. Zero installs. 100% private. Directly in your browser.**

Live Demo → https://ubiquitous-griffin-223f29.netlify.app

### **The Problem**
Creators in Angola and emerging markets can't afford CapCut Pro, Adobe, or gaming PCs. Editing video means lag, crashes, and privacy risks sending files to the cloud.

### **The Solution**
Gelson IA is a browser-based AI studio powered by WebAssembly + MediaPipe. 
- **Real-time filters**: Beauty, lighting, background removal
- **Zero installs**: Works on Android, iPhone, any browser
- **100% client-side**: Your video never leaves your device. No servers.
- **Export HD**: Download instantly

### **Why This Matters**
Built from Luanda, Angola 🇦🇴 for the next billion creators. 

If it runs on a cheap Android with 3G, it runs anywhere.

### **Tech Stack**
- HTML5 + JavaScript ES6
- WebAssembly for AI inference
- MediaPipe for real-time segmentation
- Client-side FFmpeg.wasm for export
